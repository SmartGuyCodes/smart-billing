# Smart Billing

Smart Billing is a modern billing and invoicing solution designed for flexibility, automation, and seamless integration into your business workflows.

## Features

- Fast and intuitive invoice creation
- Automated recurring billing
- Customer and product management
- Payment tracking and reporting
- Integration-ready API
- Secure and scalable architecture

## Getting Started

1. **Clone the repository:**

    ```bash
        git clone https://github.com/SmartGuyCodes/smart-billing.git
        cd smart-billing
    ```

2. **Install dependencies:**

    ```bash
        composer install
        npm install
    ```

3. **Configure environment:**
    - Copy `.env.example` to `.env` and update your settings.

4. **Run migrations:**

    ```bash
        php artisan migrate
    ```

5. **Start the development server:**

    ```bash
        php artisan serve
    ```

## Usage

- Access the dashboard at [http://localhost:8000](http://localhost:8000)
- Use the API for integration with external systems
